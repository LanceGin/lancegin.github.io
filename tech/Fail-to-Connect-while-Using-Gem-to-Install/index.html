TIMEOUT error occured while using Gem to Install

Aug 19, 2015 | tech

When I used Rubygems to install some application times ago, I usually meet the problem that says “connection error”.

Today when I tapped sudo gem install rails, I met the problem again, just like the pic below.

Now I introduce an easy way to solve this problem.

• use gem sources to list the sources you have used. You will see the source https://rubygems.org/

• use gem sources -a http://rubygems.org to add the new source with “http”

• use gem sources -r https://rubygem.org/ to remove the old source with “https”

• then you will not have the connection problem.

• If you still have some problem when you use bundle, you can use gem sources -a http://ruby.taobao.org to add taobao ruby to your source.
