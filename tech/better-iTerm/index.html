Useful Guide about Iterm and Zsh

in this article, I will set an very easy Iterm, the plugins I will use are very covenient for us to use the Iterm, Vim ,and Terminal.

Dowload Iterm

click iterm2.com to download the latest version.

Install Zsh

1. via curl

curl -L https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh | sh

2. via wget

wget https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O - | sh

3. manual installation

. clone the repository:

git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh

. backup your ~./zshrc file

cp ~/.zshrc ~/.zshrc.orig

. creat your new configuration file

cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc

. change your default shell

chsh -s /bin/zsh

now you have finish installing your new toys–iTerm2 and oh-my-zsh. You can just have a try.

Font Setting

  1. open your iTerm2, and open the Profiles or press command + O
  2. select your loving font.

Blendent

if you use iTerm2 on Mac, you will find that the default blendent is very ugly. I can’t stand it.

so taday I will recommend a very beautiful color blendent Solarized

like this shot below

  1. download Salarized

git clone git://github.com/altercation/solarized.git

  1. add the color blendent to your iterm2

open the folder you downloaded just now, then select the folder iTerm2-colors-solarized, double click the .itermcolor file, now you can open your Profiles to select your new blendent.

now, I found another beautiful blendent named “tomorrow”,you can download it via git clone git:github.com/chriskempson/tomorrow-theme, then goto step 2 above.

Theme

. oh-my-zsh has provided many kinds of Themes, choose the best one you like.

. open your .zshrc file

vi ~/.zshrc

. find ZSH_THEME="agnoster", and change the theme name.

. reopen iTerm2.
